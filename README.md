# Архиватор на базе алгоритма Хаффмана

Программа на языке C++, реализующая сжатие и распаковку файлов с использованием алгоритма Хаффмана.

## Сборка

```bash
cmake -B build
cmake --build build
```

После сборки исполняемый файл будет находиться в папке `build`.

## Использование

### 1. Сжатие файла

```bash
./build/huffman_archiver -c <исходный_файл> <архив.huff>
```

Пример:

```bash
./build/huffman_archiver -c text.txt text.huff
```

### 2. Распаковка архива

```bash
./build/huffman_archiver -d <архив.huff> <выходной_файл>
```

Пример:

```bash
./build/huffman_archiver -d text.huff restored.txt
```

### 3. Тестовый режим

Запуск тестирования скорости работы и степени сжатия:

```bash
./build/huffman_archiver -b <файл_для_теста>
```

Пример:

```bash
./build/huffman_archiver -b text.txt
```

## Возможности

* Сжатие файлов алгоритмом Хаффмана.
* Распаковка ранее созданных архивов.
* Оценка эффективности сжатия и производительности программы.
